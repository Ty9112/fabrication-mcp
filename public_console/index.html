      EST Jobs

      View estimate jobs loaded into the EST pipeline SQLite database via the MCP tools (est_list_jobs, est_job_summary).

      EST MCP Endpoint Not Detected

      This panel demonstrates the EST-jobs UI pattern. Browsers cannot speak MCP's stdio/SSE transports directly, so the panel probes for a plain JSON-RPC-over-HTTP endpoint at localhost:8005/mcp/v1 — a thin proxy in front of the server's tools/call. With the standard setup, query EST jobs from your MCP client instead (est_list_jobs, est_job_summary).
